## 📝 Recommended GitHub Description

```
🚀 Signalist - Real-Time Stock Market Intelligence Platform

A full-stack Next.js 15 application for retail investors featuring live market 
charts, personalized watchlists, AI-powered insights, and daily automated news 
summaries. Built with TypeScript, MongoDB, Tailwind CSS, and powered by Finnhub 
API and Google Gemini AI.

Features:
• Real-time stock dashboard with TradingView widgets
• Advanced stock search (Cmd/Ctrl + K)
• Personalized watchlists with live price updates
• AI-powered market insights using Google Gemini
• Daily automated email market summaries
• Email/password authentication
• Dark-themed, responsive UI

Tech Stack:
Next.js 15 • TypeScript • React 19 • MongoDB • Tailwind CSS • 
Finnhub API • Google Gemini • Nodemailer • Better Auth • Inngest

🌐 Production Ready: Deployment optimized for Vercel, Railway, AWS, Docker
📖 Docs: See SETUP.md for local development and DEPLOYMENT.md for deployment
```
